## 八、三类记忆创建与分类测试

> **分类标准说明**：
> - **情景记忆 (episodic)**: 会话中发生的重要事件，如"用户请求迁移数据库"、"Plan 完成"
> - **语义记忆 (semantic)**: 关于项目、代码、用户偏好的知识，如"项目使用 Spring Boot 3.4 + MySQL"
> - **程序记忆 (procedural)**: 成功的操作模式或工作流，如"部署流程: mvn package → docker build → kubectl apply"
>
> **创建路径**：
> 1. **巩固提取** — token 使用率 >75% 触发巩固时，LLM 自动从旧对话中提取三类记忆
> 2. **Plan 完成** — Plan 所有步骤完成后自动创建 1 条 episodic + N 条 procedural 记忆

### TC-MEM-01: 巩固产生的情景记忆

| 项 | 内容 |
