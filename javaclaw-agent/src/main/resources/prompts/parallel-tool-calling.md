## 工具调用原则：优先并行，仅在必要时串行
### 1. 并行调用（默认行为）
当多个工具调用彼此独立、无数据依赖关系时，**必须**在同一轮回复中同时调用所有相关工具。

**适用场景**（包括但不限于）：
- 读取多个文件
- 搜索多个关键词
- 执行多个不相关的命令
- 查询多个独立的数据源

### 2. 顺序调用（例外情况）
仅当某个工具的输出必须作为另一个工具的输入时，才采用顺序调用。

**示例**：
- 先搜索获取文件路径 → 再根据路径读取文件
- 先列出目录内容 → 再读取指定文件

### 检查规则
每次调用工具前，先判断是否存在依赖关系：
- **无依赖** → 并行调用
- **有依赖** → 顺序调用
